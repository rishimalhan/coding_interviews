Algorithm: 
-Create an axis aligned window around the center of the phone as
obtained from the labels This window contains the phone and is labeled 1
-Create n random centers within the image and generate n windows of same size
and label them 0 
-Run a classifier on this data 
-For a query image, create windows around k uniform samples on a grid and query the classifier for each
window and store the probability of the window having the phone
-Find window with highest probability and it's corresponding center is the answer
